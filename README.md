# Proto Shadcn

A rapid prototyping platform. Describe a feature, see it built in real time.

## Getting Started

1. **Clone the repo**
   ```bash
   git clone <repo-url>
   cd proto-shadcn
   ```

2. **Open Claude Code** (or Cowork) in this folder

3. **Start prompting** — describe the feature you want to prototype. Claude will handle the rest (installing dependencies, starting the server, building the UI).

That's it. No manual setup required.

## How It Works

Each feature is built as an isolated **module** with its own pages, components, and data. A module selector in the header lets you switch between prototypes.

## Stack

React, TypeScript, Tailwind CSS, shadcn/ui, Vite
